# Rev 100

간단한 xor 문제이다.

```c
int __cdecl main(int argc, const char **argv, const char **envp)
{
  size_t v4; // rax
  size_t v5; // rax
  int i; // [rsp+Ch] [rbp-4h]

  std::operator<<<std::char_traits<char>>(&std::cout, "try to find me :D\n", envp);
  if ( i_am_debugged() != 0 )
    return 1;
  v4 = strlen(A);
  memcpy(&dest1, A, v4 + 1);
  if ( i_am_debugged() != 0 )
    return 1;
  v5 = strlen(B);
  memcpy(&dest2, B, v5 + 1);
  for ( i = 0; i < 22; ++i )
  {
    if ( i_am_debugged() != 0 )
      return 1;

```

ptrace 안티디버깅 무시하고 플래그 구할 수 있다.

C[i] 배열에 A[i] ^ B[i]를 넣고있다. 그냥 테이블 값 가져와서 xor 연산해주면 된다.
